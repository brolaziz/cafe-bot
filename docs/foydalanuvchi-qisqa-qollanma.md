# Buyurtma va to‘lov — qisqa qo‘llanma

## 1. Menyu ochish

Botda **/start** bosing yoki **«Menyuni ochish»** tugmasi orqali mini-ilovani oching.

## 2. Buyurtma berish

1. Mahsulotlarni tanlang, **savat**ga qo‘shing.  
2. **Buyurtma** / checkout bo‘limiga o‘ting.  
3. Telefon va manzil (yoki kafedan olib ketish)ni to‘ldiring.  
4. Buyurtmani **yakunlang**.

## 3. To‘lov (P2P)

1. Buyurtma yaratilgach ekranda **karta raqami** va **egasi** ko‘rinadi — kerak bo‘lsa **nusxa** oling.  
2. Banking ilova yoki boshqa usul bilan **shu kartaga** buyurtma summasini o‘tkazing.  
3. To‘lovdan keyin **chek** yoki to‘lovni tasdiqlovchi **rasm** tayyorlang (skrinshot ham bo‘ladi).

## 4. Chek yuborish

1. Pastki menyudan **Profil** (yoki buyurtma tugagach chiqadigan **Buyurtmalarimga o‘tish**) oching.  
2. **Buyurtmalar** ro‘yxatidan shu buyurtmani toping.  
3. **Chek yuklash** tugmasidan foydalanib rasmni yuboring.

Chek adminga boradi; tasdiqlangach buyurtma tayyorlanadi. Holat shu yerda yangilanib turadi.

## 5. Tayyor bo‘lishi

To‘lov tasdiqlangach va buyurtma tayyor bo‘lgach sizga bot orqali **xabar** kelishi mumkin. Ilovani vaqt-vaqtida tekshirib turing.

---

**Eslatma:** to‘lovni amalga oshirish va chekni yuborishni unutmang — aks holda buyurtma kutilishda qoladi.
